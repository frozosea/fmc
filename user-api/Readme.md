## User api microservice
### How to run?
#### Set env variables

- POSTGRES_USER
- POSTGRES_PASSWORD
- POSTGRES_DATABASE
- POSTGRES_HOST
- POSTGRES_PORT
- REDIS_URL
- GRPC_PORT
- REDIS_TTL
- JWT_SECRET_KEY
- SENDER_NAME
- SENDER_EMAIL
- UNISENDER_API_KEY

#### Go to `conf/cfg.ini` and change parameters if you want
