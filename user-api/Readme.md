## User api microservice

### How to run:

    make run 

### Env variables

- POSTGRES_USER
- POSTGRES_PASSWORD
- POSTGRES_DATABASE
- POSTGRES_HOST
- POSTGRES_PORT
- REDIS_URL
- GRPC_PORT
- REDIS_TTL
- JWT_SECRET_KEY
- ADMIN_ACCESS_KEY
- SEND_EMAIL 
- PASSWORD 
- SMTP_HOST 
- SMTP_PORT 
- SEND_TO_EMAILS

Microservice methods are in `pkg/proto`.
